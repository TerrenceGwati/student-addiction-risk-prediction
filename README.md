# student-addiction-risk-prediction

This project explores the use of machine learning to predict student drug addiction risk using academic, behavioral, and social indicators.

## Project Structure

student-addiction-risk-prediction/
│
├── data/
│ ├── raw/
│ │ ├── student_addiction_dataset_train.csv
│ │ └── student_addiction_dataset_test.csv
│ └── processed/
│
├── notebooks/
├── reports/
├── src/
├── .gitignore
├── requirements.txt
└── README.md


## Data

- **data/raw/student_addiction_dataset_train.csv**  
  The original training dataset; contains labeled examples used to train and validate the predictive model.

- **data/raw/student_addiction_dataset_test.csv**  
  The original test dataset; used for final model evaluation (unseen during training).
